# pybibletype
A Bible typer written in Python. Can also display your progress in a few different ways.

This project cannot show the text you need to enter, and it cannot figure out where you made mistakes, because it does not have access to the text of the Bible. It only knows how many verses are in each chapter and how many chapters are in each book.

Projects are stored in the "projects" folder. Included is an "NIV Bible" project. It only says NIV in case a different translation has different numbers of chapters and verses, because this project was created by looking through an NIV Bible. Don't worry - *it contains no text from the NIV Bible, or from any Bible for that matter.*

More information will appear here later, and the Wiki should become a user's manual, once I actually write all of that.
